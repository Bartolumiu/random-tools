# WebP to PNG Converter

Mass convert WebP images to PNG format.

## Requirements
- Python 3.x
- Pillow (`pip install Pillow`)

## Usage
Set your input and output directories in the script or modify the defaults, then run:
```pwsh
python webp2png.py
```
Defaults:
- Input folder: `in`
- Output folder: `out`
